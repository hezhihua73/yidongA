股票异动检测系统使用说明
========================

一、系统要求：
- Windows 7/8/10/11 操作系统
- 确保电脑能够连接互联网

二、首次使用步骤：
1. 双击运行 "安装依赖库.bat" 文件，安装必要的Python库
2. 安装完成后，双击运行 "启动股票异动检测系统.bat" 文件启动程序

三、程序使用方法：
1. 运行程序后，按照提示输入6位股票代码（如 002149）
2. 程序会自动检测该股票的异动情况
3. 输入 'help' 可查看帮助信息
4. 输入 'quit'、'exit'、'q' 或 '退出' 可退出程序

四、功能说明：
- 检测连续10个交易日内，涨跌幅偏离值累计达 +100% 的股票
- 检测连续30个交易日内，涨跌幅偏离值累计达 +200% 的股票
- 偏离值计算公式：单只股票涨跌幅 - 对应指数涨跌幅
- 新增功能：显示临界价格和临界涨幅，帮助用户了解触发异动所需的条件

五、注意事项：
- 程序需要网络连接以获取实时股票数据
- 如果出现错误提示，请确保网络连接正常
- 首次运行可能需要稍长的时间来获取数据
- 程序会自动处理网络连接失败等异常情况
- 如果双击运行出现乱码，可以右键点击 "启动股票异动检测系统.bat" 文件，
  选择 "编辑"，然后按Ctrl+Shift+Esc打开任务管理器，
  或者直接在命令提示符中运行该文件

六、退出程序：
- 输入 'quit'、'exit'、'q' 或 '退出' 均可退出
- 退出时会提示按回车键，按任意键即可关闭窗口

七、解决乱码问题：
- 如果运行时出现中文乱码，建议在命令提示符（cmd）中运行程序
- 打开方法：按Win+R，输入cmd，回车，然后输入命令运行程序
- 或者运行 "启动股票异动检测系统.bat" 批处理文件
- 程序已添加中文编码支持，应该可以正常显示中文字符

八、解决文件缺失问题：
- 如果遇到 FileNotFoundError 或 [Errno 2] No such file or directory 错误
- 这是由于akshare库依赖文件未正确打包导致的
- 我们已使用 --collect-data akshare 参数重新打包，应该已解决此问题
- 如果仍有问题，可以尝试重新运行 "安装依赖库.bat" 重新安装akshare库
